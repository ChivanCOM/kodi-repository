# ChivanCOM Repository for Kodi

This is the ChivanCOM Kodi addon repository. Once installed, Kodi will automatically receive updates for all included addons.

## Installation

Follow the step-by-step installation guide at:

**[https://github.com/ChivanCOM/kodi-repository-install](https://github.com/ChivanCOM/kodi-repository-install)**

## Addons included

| Addon | Description |
|-------|-------------|
| **iBroadcast** (`plugin.audio.ibroadcast`) | Stream your iBroadcast music library in Kodi |
| **Album Art Visualizer** (`plugin.visualization.albumart`) | Full-screen album art visualizer with animated backgrounds |
